Questions and Answers

1. Why is Git not showing the git commit command while adding a file and showing the command for unstage?

Git shows the unstage command when files are already staged. Use git commit after staging. If files are not staged, use git add to stage them first. Ensure you are in the correct directory.

2. What is the difference before the first push and after the first push?

Before the first push, the branch is local. After the first push, it is tracked by the remote repository. This allows collaboration and version control with others. Changes can be pulled and pushed.

3. How is the title tag fetched to the head and h1 fetched to the body when rendering an HTML file which has no structure?

The browser automatically places the <title> tag in the <head> and <h1> in the <body> if not structured. This ensures proper document structure and rendering. It follows HTML standards.

4. What is the difference between href and src and why is it used in img and anchor tags?

href specifies the URL for a link, while src specifies the source for an embedded resource like an image. href is used in anchor tags, and src is used in image tags. They define the destination and source respectively.

5. Why are there three dots in if...else and do...while?

The three dots in if...else and do...while indicate omitted code or logic. They are placeholders for additional code that can be added later. It signifies continuation or repetition.

6. What is the difference between spread and rest operators?

The spread operator expands elements, while the rest operator collects multiple elements into an array. Spread is used in function calls, and rest is used in function parameters. They handle elements differently in arrays and objects.
