Interview Questions

1. What is the name of the tree in DOM?

The tree in the Document Object Model (DOM) is called the DOM tree. It represents the structure of a document as a tree of nodes.

2. What is Git?

Git is a distributed version control system that tracks changes in source code during software development. It allows multiple developers to work on a project simultaneously.

3. If you want to make a folder as a repository of your github then you need to do what?

To make a folder a repository on GitHub, you need to initialize it with git init, add your files with git add ., commit them with git commit -m "Initial commit", and then link it to a GitHub repository with git remote add origin and git push -u origin master.

4. Why git is fast in executing the commands typed?

Git is fast in executing commands because it uses a local repository for most operations, reducing the need for network communication. It also employs efficient algorithms for data storage and retrieval.

5. What is the staging area called as?

The staging area in Git is also called the index. It is used to prepare changes before committing them to the repository.
