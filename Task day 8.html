Question 1: Why are multiple spaces ignored in a textarea?

Answer: In HTML, multiple spaces are ignored in a textarea because HTML collapses whitespace by default. To preserve multiple spaces, you can use the white-space CSS property.

Question 2: How does handleEvent work without sending any arguments?

Answer: The handleEvent method is part of the EventListener interface. When an event occurs, the event object is automatically passed to the handleEvent method.

Question 3: How is component code converted to vanilla HTML code?

Answer: Component-based frameworks like React or Vue.js use a build process to convert component code into vanilla HTML, CSS, and JavaScript.
